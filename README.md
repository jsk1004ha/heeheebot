# HeeHeeBot

디스코드 종합 봇의 초석입니다. 현재 구현된 기능은 **레벨/돈, 운세, 끝말잇기, RPG, 검 강화/판매/랭킹/배틀, 도박, 가상주식, 관리 시스템**입니다.

## 현재 포함된 기능

- 메시지 활동 보상
  - 일반 채팅마다 5~15 XP 지급
  - 하루 첫 채팅은 추가 50 XP 지급
  - 필요 경험치 공식: `floor(100 × 레벨^1.5)`
  - 경험치가 충분하면 자동 레벨업
  - 레벨업 보너스 지급
- `/프로필`
  - 내 레벨, 경험치, 연속 출석, 보유금 확인
- `/출석`
  - 하루 1회 100 XP + 랜덤 1~1,000 코인 지급
  - 연속 출석 3/5/7/14/30일 배수에 추가 XP 지급
- `/송금`
  - 다른 유저에게 돈 송금
- `/랭킹`
  - 서버 내 레벨/경험치 기준 TOP 랭킹
- `/운세`
  - 한국시간 기준 오늘운세/어제운세/내일운세 확인
  - 운세 분류: `大吉(대길) → 吉(길) → 中吉(중길) → 小吉(소길) → 末吉(말길) → 凶(흉) → 大凶(대흉)`
  - 총 1,400개 운세 문구를 7단계별 200개씩 사용
  - 같은 서버, 같은 유저, 같은 날짜의 운세는 누가 보더라도 항상 동일
  - `대상` 옵션으로 다른 유저 운세 확인 가능
  - 운세 확인 보너스 XP는 하루 1회 지급
- RPG 전투
  - `/rpg 시작 직업 성별`
    - 옵션을 비우면 직업+성별 버튼으로 캐릭터 생성/변경
    - 초보자/전사/마법사/궁수 기본 직업 선택
    - 직업별 남캐/여캐 외형 선택
    - 팔라딘/도적/사제는 가챠로 해금 후 선택
    - 시작/상태/전투/보스/탐험 응답은 Discord embed형 RPG 카드로 이미지 표시
  - `/rpg 메뉴`
    - RPG 메인 허브에서 레벨 진행바, 전투력, 현재 목표, 추천 행동, 오늘 의뢰 요약 확인
    - 전투/탐험/던전/일일 의뢰/월드맵을 버튼으로 바로 실행
  - `/rpg 전투 난이도 지역 스킬`
    - 쉬움/보통/어려움 몬스터와 1회 전투
    - 지역 선택 가능: 초록 숲, 수정 동굴, 그림자 늪, 고대 유적, 화산 협곡, 빙결 봉우리, 하늘 성채
    - 지역을 비우면 `/rpg 지역`에서 이동한 현재 지역 사용
    - 직업별 스킬을 사용하면 MP를 소모하고 전투력이 상승
    - 승리 시 RPG XP와 코인 지급
    - 등급/랜덤 옵션이 붙은 몬스터 드랍 장비 획득 가능
    - 전투 기록과 60초 쿨다운 저장
  - `/rpg 탐험 지역`
    - 지역별 랜덤 이벤트: 몬스터 조우, 보물상자, 함정, 성소
    - 보상, 피해, 드랍 장비, 이미지 배경을 함께 반환
  - `/rpg 던전 지역 깊이`
    - 1~5층 연속 탐험으로 전투/보물/함정 이벤트를 누적 정산
    - 클리어 층수, 획득 XP/돈, 드랍 장비를 저장
  - `/rpg 보스`
    - 슬라임 킹, 고대 드래곤 같은 보스에게 수동 턴제 전투로 도전
    - 공격/직업 스킬/방어/포션 버튼으로 직접 행동 선택
  - `/rpg 상태`
    - 내 RPG 직업, 성별, 전직, 직업 숙련, 해금 직업, 지역 탐사율, HP/MP, 스탯, 전적, 승률, 스킬 포인트, 스토리, 도감 확인
  - `/rpg 상점`
    - 회복 포션, 무기, 방어구, 장신구 구매
  - `/rpg 인벤토리`
    - 보유 아이템, 장착 장비, 랜덤 드랍 장비, HP, 공격력/방어력 확인
  - `/rpg 사용`
    - 회복 포션 같은 소비 아이템 사용
  - `/rpg 장비`
    - 보유 장비를 버튼으로 확인하고 무기/방어구/장신구 장착
  - `/rpg 전리품`
    - 탐험/전투/던전/레이드에서 얻은 랜덤 장비를 번호/버튼으로 장착
  - `/rpg 퀘스트`
    - 전투 조건을 달성하고 XP/돈/아이템 보상 수령
  - `/rpg 일일`
    - 오늘의 RPG 의뢰판 확인
    - 전투/승리/탐험/던전 일일 목표를 달성하면 XP/돈/아이템 보상 수령
    - 보상 가능한 의뢰는 버튼으로 즉시 수령 가능
  - `/rpg 휴식`
    - HP와 MP를 최대치까지 회복
  - `/rpg 가챠`
    - 돈을 소모해 고급 장비와 고급 직업 해금
  - `/rpg 스킬트리`
    - 레벨 기반 스킬 포인트로 무기숙련, 강철육체, 마나순환, 직업숙련 노드 습득
  - `/rpg 전직`
    - 옵션을 비우면 전직 트리와 전직 가능 버튼 표시
    - 레벨, 직업 숙련, 전직 퀘스트 조건을 모두 만족해야 전직 가능
    - 레벨/직업 조건을 만족하면 버서커, 아크메이지, 스나이퍼, 크루세이더, 섀도우, 세인트로 전직
  - `/rpg 스토리`
    - 숲/동굴/늪/유적/화산/빙결/하늘 성채 챕터를 지역 진행도와 보스/던전/레이드 조건에 따라 진행
  - `/rpg 도감`
    - 처치한 몬스터/보스의 도감 보상을 1회 수령
  - `/rpg 레이드`
    - 서버 RPG 성장용 레이드 보스 도전, 대량 보상과 전설급 드랍 가능
  - `/rpg 지역`
    - 월드맵, 지역 해금 조건, 보상 배율 확인
    - 지역별 탐사율을 확인하고 전투/탐험/던전/보스 진행으로 탐사율 상승
    - 해금된 지역은 버튼 또는 `지역` 옵션으로 이동
- 검 강화/판매/랭킹/배틀
  - `/검강화`
    - 기존 보유금을 사용해 검을 최대 +100강까지 강화
    - 결과는 강화/유지/파괴이며, 파괴 시 검은 +0으로 초기화
    - +20 이상에서 파괴되면 제련석을 일부 돌려받음
    - 강화 결과마다 대장장이가 축하/위로/놀림 메시지를 남기고 성공/유지/파괴별 대장장이 이미지가 함께 표시됨
    - +50/+80/+100 고강화 성공과 +50/+80 이상 파괴에는 특수 대장장이 멘트 표시
  - `/검상급강화`
    - 제련석과 보유금을 사용해 파괴 없이 강화
    - +90 이하에서만 시도 가능
  - `/검정보`
    - 현재 검, 최고 강화, 다음 일반/상급 강화 비용·확률, 판매 예상가, 배틀 잔여 횟수 확인
  - `/검판매`
    - 현재 강화된 검을 판매해 봇 내부 돈을 받음
    - 판매 금액은 해당 강화 단계까지의 일반 강화 최소 비용 합계의 60%
    - 판매 전 `판매 확정`/`취소` 버튼으로 한 번 확인
    - 판매 후 현재 검은 +0으로 초기화되고, 최고 강화 기록/제련석/배틀 전적은 유지
  - `/검랭킹`
    - 최고강화, 판매금, 파괴횟수 기준 서버 랭킹 확인
  - `/선물받기`
    - 하루 한 번 상급강화 재화인 제련석 3개 수령
  - `/검배틀`
    - 상대를 비우면 랜덤 검배틀 진행
    - 상대 유저를 지정하면 수락 버튼으로 유저 간 검배틀 진행
    - 하루 10회 제한, 승자는 돈/경험치/제련석을 얻고 패자는 돈을 잃지 않음
- 도박/미니게임
  - `/끝말잇기 시작`
    - 1분 동안 버튼으로 참가자를 모집한 뒤 시작
    - 모드 선택 가능: `일반`, `매너(한방단어 금지)`, `쿵쿵따(3글자만)`
    - `희희봇참가:false`로 AI 없이 유저끼리도 진행 가능(모드와 독립)
    - 참가자와 희희봇 AI 또는 참가자끼리 순서대로 단어를 입력
    - 자기 차례에 처음 30초 안에 답하지 못하면 탈락
    - 성공 단어가 나올 때마다 제한시간이 1초씩 줄어들며 최소 5초까지 감소
    - 최종 유저 승자는 보유금 보상 지급, 희희봇이 이기면 마지막으로 탈락한 준우승 유저에게 상금 지급
    - 모든 참가자는 생존 순위에 따라 XP 지급(빨리 탈락할수록 적음)
    - 끝말잇기 XP로 레벨업하면 기존 레벨업 보너스 돈도 함께 지급
    - 단어 DB는 AutoKkutu `KkutuDbDump/kor_list.txt` 기반 357,644개 한국어 단어(`data/wordchain-ko.txt`) 사용
  - `/홀짝 돈 선택`
    - 홀/짝 적중 시 약 1.9배 지급
    - 99~100은 하우스 승리라 실제 승률은 약 49%
  - `/주사위 돈 선택`
    - 높음(4~6) 또는 낮음(1~3) 적중 시 2배 지급
  - `/슬롯 돈`
    - 낮은 확률로 3배, 10배, 20배 지급
  - `/블랙잭 돈`
    - 상대를 비우면 AI와 버튼으로 수동 대결
    - 히트/스탠드를 직접 선택
    - 승리 시 1.5배, 블랙잭 승리 시 2배, 무승부는 환불
  - `/블랙잭 돈 상대`
    - 상대 유저가 버튼으로 수락하면 수동 대결 시작
    - 양쪽 유저가 차례대로 히트/스탠드를 선택
    - 양쪽 잔액을 검사한 뒤 같은 금액을 걸고 승자가 판돈 전체를 가져감
  - `/룰렛 돈 선택`
    - 빨강/검정/홀/짝/낮음/높음/0 베팅
  - `/바카라 돈 선택`
    - 플레이어/뱅커/타이 베팅
  - `/크랩스 돈 선택`
    - 패스/돈패스 라인 베팅
  - `/시크보 돈 선택`
    - 세 주사위의 작음/큼/트리플 베팅
	  - `/키노 돈 번호들`
	    - 1~80 중 1~5개 번호 선택, 10개 추첨 번호와 비교
- 가상주식

  - `/주식 레버리지진입 종목 방향 배율 증거금`
    - 1~100배 롱/숏 가상 레버리지 포지션 진입
  - `/주식 레버리지청산 포지션`

  - `/주식 레버리지보유`
    - 레버리지 포지션, 평가금, 미실현손익, 자동 청산 내역 확인

## 시작하기

Node.js 22.5 이상이 필요합니다. SQLite 저장소는 Node.js 내장 `node:sqlite` 모듈을 사용합니다.

```bash
npm install
cp .env.example .env
```

`.env`에 봇 토큰과 애플리케이션 ID를 넣은 뒤:

```bash
set -a
source .env
set +a
npm run register
npm start
```

> 채팅 XP 자체는 메시지 이벤트만 있으면 동작하지만, 금칙어 감지와 자동 도배 대응은 메시지 내용을 확인합니다.
> Discord Developer Portal에서 **Message Content Intent**를 켜야 금칙어/도배 감지가 정상 동작합니다.

## 서버 관리 예시

```text
/관리설정 로그채널 #로그
/관리설정 금칙어추가 나쁜말
/관리설정 경고누적처벌 3 뮤트 10분
/경고 @유저 도배
/뮤트 @유저 10분 도배
/청소 30
```

봇 역할은 관리 대상 유저보다 위에 있어야 하며, 필요한 권한은 `메시지 관리`, `멤버 타임아웃`, `킥`, `밴`, `서버 관리`입니다.

## 끝말잇기 명령어 예시

```text
/끝말잇기 시작
/끝말잇기 시작 모드:매너(한방단어 금지)
/끝말잇기 시작 모드:쿵쿵따(3글자만)
/끝말잇기 시작 희희봇참가:false
```

끝말잇기는 한 채널당 하나만 진행됩니다. 모집 메시지의 `참가` 버튼으로 참가하면 1분 뒤 시작됩니다. 희희봇을 참가시키면 유저 1명부터 시작할 수 있고, 희희봇 미참가 게임은 유저가 최소 2명 필요합니다. 매너 모드는 다음 사람이 이을 수 없는 한방단어를 금지하고, 쿵쿵따는 정확히 3글자 단어만 허용합니다.

## 운세 명령어 예시

```text
/운세
/운세 날짜:오늘운세
/운세 날짜:어제운세
/운세 날짜:내일운세
/운세 대상:@유저
/운세 날짜:어제운세 대상:@유저
```

운세 결과는 사용자 ID와 날짜를 기준으로 고정됩니다. 그래서 다른 사람이 내 운세를 확인해도 내가 직접 보는 운세와 같은 문구가 나옵니다.
운세 분류는 `大吉(대길)`, `吉(길)`, `中吉(중길)`, `小吉(소길)`, `末吉(말길)`, `凶(흉)`, `大凶(대흉)` 7단계로 표시되며, 각 단계마다 200개씩 총 1,400개 문구가 준비되어 있습니다.
운세 확인 XP는 한국시간 기준 하루 한 번만 받을 수 있습니다.

## 검 강화 명령어 예시

```text
/선물받기
/검강화
/검상급강화
/검정보
/검판매
/검랭킹 종류:최고강화
/검랭킹 종류:판매금
/검랭킹 종류:파괴횟수
/검배틀
/검배틀 상대:@유저
```

일반 검강화는 돈을 사용하며 강화/유지/파괴가 발생합니다.
상급강화는 제련석을 사용하고 파괴되지 않지만 +90 이하에서만 가능합니다.
검강화와 검상급강화 결과에는 성공/유지/파괴별 대장장이 NPC의 축하/위로/놀림 메시지와 썸네일이 함께 표시됩니다.
검판매는 확인 버튼을 누른 뒤 현재 검을 판매해 돈을 받고 현재 검 레벨만 +0으로 되돌립니다.
검랭킹은 최고강화, 판매금, 파괴횟수 기준으로 확인할 수 있습니다.
검배틀은 하루 10회까지 가능하며, 패배해도 돈을 잃지 않습니다.

## 도박 명령어 예시

```text
/홀짝 100 홀
/주사위 100 높음
/슬롯 100
/블랙잭 500
/블랙잭 500 @상대
/룰렛 100 빨강
/바카라 100 뱅커
/크랩스 100 패스
/시크보 100 큼
/키노 100 3,14,25
```

모든 도박 기능은 봇 내부 가상 돈만 사용합니다. 실제 돈 결제/환전 기능은 없습니다.
블랙잭은 자동 정산이 아니라 버튼으로 히트/스탠드를 직접 선택합니다.
유저 간 블랙잭은 상대가 수락해야만 진행되며, 시작 전과 정산 시 양쪽 잔액을 검사합니다.

## 가상주식 명령어 예시

```text
/주식 시세
/주식 전체시세
/주식 시세 종목:<자동완성에서 원숭이닉스 선택>
/주식 매수 종목:<자동완성에서 희진전자 선택> 수량:10
/주식 매도 종목:<보유 종목 자동완성에서 희진전자 선택> 수량:5
/주식 보유
/주식 랭킹
/주식 레버리지진입 종목:<자동완성에서 원숭이닉스 선택> 방향:롱 배율:100 증거금:10000
/주식 레버리지진입 종목:<자동완성에서 도훈건설 선택> 방향:숏 배율:50 증거금:5000
/주식 레버리지보유
/주식 레버리지청산 포지션:0-1
```

가상주식은 봇 내부 가상 돈만 사용하는 디코밈 게임입니다. 실제 주식, 실제 투자,
실제 돈 결제/환전 기능과 연결되지 않습니다. 36개 가상 종목은 3분 단위 tick으로
가격과 밈 뉴스가 갱신되며, 현물 매수/매도에는 1% 수수료가 적용됩니다. 시세에는
상승(🔴 ▲), 하락(🔵 ▼), 보합(⚪ —) 색상 표시가 붙고, 종목 입력은 디스코드
자동완성 선택을 지원합니다.
레버리지는 빚을 만들지 않는 가상 파생 포지션이며 1~100배 롱/숏을 지원합니다.
레버리지 진입 수수료는 증거금의 2%이고, 손실이 증거금 100%에 도달하면 자동 청산됩니다.

## RPG 명령어 예시

```text
/rpg 시작
/rpg 시작 직업:전사 성별:여캐
/rpg 메뉴
/rpg 지역
/rpg 지역 지역:그림자 늪
/rpg 전투 난이도:보통 스킬:파워 스트라이크
/rpg 탐험
/rpg 던전 지역:수정 동굴 깊이:3
/rpg 보스 보스:슬라임 킹
/rpg 상점
/rpg 상점 아이템:철검 수량:1
/rpg 장비
/rpg 장비 아이템:철검
/rpg 전리품
/rpg 전리품 장비:1
/rpg 인벤토리
/rpg 사용 아이템:회복 포션
/rpg 퀘스트
/rpg 일일
/rpg 일일 임무:승리 계약
/rpg 휴식
/rpg 가챠 배너:일반 소환 횟수:1
/rpg 스킬트리 스킬:무기 숙련
/rpg 전직
/rpg 전직 전직:버서커
/rpg 스토리 챕터:숲의 맹세
/rpg 도감 몬스터:숲 슬라임
/rpg 레이드 레이드:슬라임 군단
/rpg 상태
```

RPG는 기존 레벨/돈 시스템을 사용합니다. 현재는 직업, 직업별 남캐/여캐 외형,
직업 숙련과 전직 퀘스트, 지역 해금과 탐사율, 몬스터 전투, 탐험, 던전,
랜덤 등급 장비, HP/MP/스탯, 직업별 스킬, 스킬트리, 전직, 수동 턴제 보스전,
레이드, 상점, 장비, 인벤토리, 퀘스트, 일일 의뢰, 추천 행동이 있는 버튼형
메인 허브, 스토리 챕터, 몬스터 도감, 가챠, 턴제 PvP, Discord embed형 RPG 카드 이미지까지 연결되어 있습니다.
