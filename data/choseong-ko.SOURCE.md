# choseong-ko.txt source

Generated by `tools/generate-choseong-words.js` from two Korean word sources:

1. Local AutoKkutu Korean KKuTu gameplay dump
   - Source file in this repo: `data/wordchain-ko.txt`
   - Upstream source: AutoKkutu `KkutuDbDump/kor_list.txt`
   - Used as the bulk Korean dictionary source for broad 초성게임 coverage.
2. Wiktionary **Appendix:Basic Korean Vocabulary List** pages
   - Source root: https://en.wiktionary.org/wiki/Appendix:Basic_Korean_Vocabulary_List
   - Source description: Korean learning vocabulary list (`한국어 학습용 어휘목록`) with ranks designated by the National Institute of Korean Language.
   - Crawled pages: root page plus `/500`, `/1000`, `/1500`, `/2000`, `/3000`, `/4000`, `/5000`, `/6000`, `/7000`, `/8000`, `/9000`, and `/Above 10,000`.

Runtime filter:

- NFC-normalized pure Hangul syllable words only.
- Exactly 2 Hangul syllables, matching the fixed 2-initial-consonant prompt.
- Unique and Korean-locale sorted.
- Generated count: 77,489 words.

The Discord 초성게임 always presents exactly 2 initial consonants (for example `ㄱㄴ`) and accepts only 2-syllable words from `data/choseong-ko.txt` at runtime. Regenerate with:

```bash
node tools/generate-choseong-words.js --output data/choseong-ko.txt
```
