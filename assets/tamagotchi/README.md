# 희진 다마고치 에셋

`tools/generate-tamagotchi-assets.js`가 `희진.png`의 색상 팔레트를 샘플링해서 만든 귀여운 도트 스타일 PNG 에셋입니다.

- 성장 단계: 알 → 유아기 → 유년기 → 청소년기 → 성년기
- 스킨: 24개
- 단계별 스킨 이미지: 120개
- 꾸미기/도구: 32개
- 원본 기반 도트 레퍼런스: `assets/tamagotchi/preview/heejin-reference-pixel.png`

재생성:

```bash
node tools/generate-tamagotchi-assets.js
```
